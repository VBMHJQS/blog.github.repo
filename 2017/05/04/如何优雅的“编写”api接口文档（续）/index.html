上传文件接口配置

　　接着昨天的bug，有一接口需要上传文件，怎样去配置注解，才能让Swagger的页面拥有测试的功能即有文件上传的入口(不知客官有咩有明白。。。)
在网上找了半天的资料也没有找到，后来我索性直接将@ApiImplicitParams注解去掉，竟然可以了，英吹思婷~，文件这块解决。
